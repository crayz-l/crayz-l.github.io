DataBase 常用函数

发表于2024-05-07|更新于2025-01-11|学习笔记
|浏览量:

数值函数

ROUND函数

格式：ROUND（value，parameter）
将value四舍五入到paramet位小数

TRUNCATE函数

格式：TRUNCATE（value，parame）
将value截断到paramet位小数

CEILING函数

格式：CELING（value）
返回大于或等于value的最小整数

FLOOR函数

格式：FLOOR（value）
返回小于或等于value的最大整数

ABS函数

格式：ABS（value）
返回绝对值

RAND函数

生成0-1之间的随机值

字符串函数

LENGTH函数

格式：LENGTH（str）
返回字符串的字符个数

UPPER函数

格式：UPPER（str）
返回大写的字符串

LOWER函数

格式：LOWER（str）
返回小写的字符串

TRIM函数

分为三种：TRIM，LTRIM，RTRIM
用于删除多余的空格

LEFT函数

格式：LEFT（str，parameter）
从左侧开始返回parameter个字符

RIGHT函数

格式：RIGHT（str，parameter）
从右侧开始返回parameter个字符

SUBSTRING函数

格式：SUBSTRING（str，start，end）
截取str的start位置到end位置，若无end则默认到str末尾

LOCATE函数

格式：LOCATE（str1，str2）
返回str1在str2第一次出现的位置

REPLACE函数

格式：REPLACE（str，str1，str2）
将str中的str1替换为str2

CONCAT函数

格式：CONCAT（str1，str2，……）
连接字符串

日期函数

NOW() 返回当前的时间
CURDATE() 返回当前的日期
CURTIME() 返回当前的时刻

EXTRACT函数

日期和时间的计算

  • 格式化日期和时间：MySQL :: MySQL 8.0 Reference Manual :: 14.7 Date and Time Functions

增加或减少日期

  • DATE_ADD(日期，INTERVAL 1 DAY) DAY可以替换为MONTH/YEAR
  • DATE_SUB(日期，INTERVAL 1 DAY) DAY可以替换为MONTH/YEAR
  • DATEDIFF(日期1，日期2)
    以日期1减去日期2，求得两个日期之间的间隔，时间间隔不纳入考虑
  • TIME_TO_SEC(时刻)
    将时刻转换为秒，从而易于计算，从凌晨0点开始计算

其他函数

IFNULL函数

格式：IFNULL(列名，字符串)
判断列值中是否有空值，若有空值，以字符串内容替换

COALESCE函数

格式：COALESCE（列名1，列名2，…，字符串）
若列名1中存在空值，以同行列名2的值进行替换，若还为空则继续替换，直至以字符串内容替换

IF函数

格式：IF（表达式，返回值1，返回值2）
表达式成立则得到返回值1，否则得到返回值2

CASE运算符

格式： CASE WHEN 表达式 THEN 返回值 …… (ELSE) 返回值 END
通过多个测试语句返回不同的结果
